3D Hover Drag Racer

This project was spawned from a love / hate of the swoop bike races in another game from my childhood, Star Wars: Knights of the Old Republic. I wanted recreate the base racing mechanic but transplant it into an infinite runner with a range of obstacles and challenges.

There are a number of additional features planned for this project including; multiple ships and maps, more types of hazards, and boss fights.

This project is currently in development and this page will continue to be updated.

All the assets used in this project were from a Synty Studios asset pack.
